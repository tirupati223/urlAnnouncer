Overview

URL Announcer is an NVDA add-on that gives blind and visually impaired users complete control over the current browser URL. Press NVDA+Shift+U to open the command layer, then press one letter to act. The URL is read silently using Windows UI Automation — your focus never moves and no keys are simulated.

Command layer entry gesture: NVDA+Shift+U
After pressing it, NVDA speaks all available commands and waits for your next key press. The layer closes automatically after 30 seconds of inactivity.

Settings Panel

Go to NVDA Menu → Preferences → Settings → URL Announcer to configure:

  • Speak URL in readable chunks — NVDA speaks it as: Protocol: HTTPS. Domain: google dot com. Path: search. Parameters: q equals nvda.
  • Automatically announce URL on each page load — Opt-in. NVDA speaks the URL every time a browser page finishes loading.
  • Maximum URLs to remember per session — Choose 5, 10, 20, or 50. History is cleared when NVDA restarts.
  • Announce all commands when layer opens — Turn off for a shorter “URL Announcer ready” message (expert mode).
  • Check for updates on NVDA startup — On by default. Silently checks for new versions and notifies you once if an update is available.

How URL Reading Works

URL Announcer reads the browser URL using Windows UI Automation (UIA) — the same accessibility API that NVDA uses internally. It asks Windows for the value of the address bar element directly. No keys are simulated, no focus moves, and your clipboard is never touched.

No focus change: Your position on the page is never disturbed. URL reading is silent and takes under 50 milliseconds.

YouTube Short Links (S key)

On any YouTube video page, pressing S converts the URL to a compact youtu.be link.

Example:
Full: https://www.youtube.com/watch?v=dQw4w9WgXcQ&t=30
Short: https://youtu.be/dQw4w9WgXcQ?t=30 (timestamp preserved)

URL History (R key)

Every URL you read, copy, or share is automatically saved to the in-memory session history. Press R to open a list. From the list you can announce the URL, copy it, or open it in the browser. The history is cleared when NVDA restarts.

Bookmarks (M and B keys)

Press M to save the current URL with a name you choose. Bookmarks are saved to %APPDATA%\nvda\urlAnnouncer_bookmarks.json and are available across NVDA restarts. Press B to open your saved bookmarks — you can open, copy, announce, or delete any entry.

Readable URL Mode

When enabled, URLs are spoken in labelled parts:

Raw:      https://www.example.com/search?q=nvda+addon
Readable: Protocol: HTTPS. Domain: example dot com.
          Path: search. Parameters: q equals nvda addon.

Security Analysis (X and D keys)

  • Protocol: HTTPS (secure) / HTTP (not secure) / file (local) / FTP
  • Domain name spoken clearly
  • Warning if the page appears to be a login or authentication page
  • Warning if the URL query string contains sensitive-sounding parameter names
  • Warning if the domain is an IP address rather than a host name
  • Warning if an unusual number of subdomains is present (phishing signal)

Installation

  1. Download urlAnnouncer-3.0.0.nvda-addon.
  2. Double-click the file. NVDA will ask to install it.
  3. Click Yes and restart NVDA when prompted.
  4. Open any browser, navigate to a webpage, and press NVDA+Shift+U.

Changing the Entry Gesture

Go to NVDA Menu → Preferences → Input Gestures → URL Announcer and assign any gesture you prefer.
