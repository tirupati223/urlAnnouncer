Overview

URL Announcer is a professional NVDA add-on that gives blind and visually impaired users complete control over the current browser URL. Press NVDA+Shift+U to open the command layer, then press one letter to act.

Command layer entry gesture: NVDA+Shift+U
This gesture was chosen because NVDA+U (without Shift) is a built-in NVDA browse-mode shortcut for navigating to the next unvisited link. NVDA+Shift+U is free of all conflicts. After pressing it, NVDA speaks all available commands and waits for your next key press.

Settings Panel

Go to NVDA Menu → Preferences → Settings → URL Announcer to configure:

  • Speak URL in readable chunks — Instead of the raw URL, NVDA speaks it as: Protocol: HTTPS. Domain: google dot com. Path: search. Parameters: q equals nvda.
  • Include page title when announcing URL — Announces "Page: Example Site. URL: https://..." when you press A.
  • Automatically announce URL on each page load — Opt-in. NVDA speaks the URL every time a browser page finishes loading.
  • Restore clipboard after reading URL — On by default. Your original clipboard content is restored after URL Announcer copies the URL internally.
  • Maximum URLs to remember per session — Choose 5, 10, 20, or 50. History is in-memory only and cleared when NVDA restarts.
  • Extended domain safety analysis — Enables IP-address detection, excessive-subdomain warnings, and other advanced checks.
  • Check for URL Announcer updates on NVDA startup — On by default. Silently checks for new versions and notifies you once if an update is available.

YouTube Short Links (S key)

On any YouTube video page, pressing S converts the URL to a compact youtu.be link.

Example:
Full: https://www.youtube.com/watch?v=dQw4w9WgXcQ&t=30
Short: https://youtu.be/dQw4w9WgXcQ?t=30 (timestamp preserved)

URL History (R key)

Every URL you read, copy, or share is automatically saved to the in-memory session history. Press R to open a list. From the list you can: announce the URL, copy it, or open it directly in the browser. The history is cleared when NVDA restarts. The maximum size is configurable in Settings.

Bookmarks (M and B keys)

Press M to save the current URL with a name you choose. Bookmarks are saved to disk (%APPDATA%\nvda\urlAnnouncer_bookmarks.json) and are available across NVDA restarts. Press B to open your saved bookmarks — you can open, copy, announce, or delete any entry.

Readable URL Mode (Settings)

When enabled, URLs are spoken in labelled parts instead of raw text:

Raw:      https://www.example.com/search?q=nvda+addon
Readable: Protocol: HTTPS. Domain: example dot com.
          Path: search. Parameters: q equals nvda addon.

Security Analysis (X and D keys)

Both X and D run the same full security check:

  • Protocol: HTTPS (secure) / HTTP (not secure) / file (local) / FTP
  • Domain name spoken clearly
  • Warning if the page appears to be a login or authentication page
  • Warning if the URL query string contains sensitive-sounding parameter names
  • Warning if the domain is an IP address rather than a host name
  • Warning if an unusual number of subdomains is present (phishing signal)

QR Code (Q key)

Pressing Q opens a QR code image of the current URL in your browser using the free qrserver.com API. No software installation required. Show the QR code to a phone camera to share the URL instantly.

How URL Reading Works

URL Announcer reads the browser URL by briefly simulating three key presses: Ctrl+L (focus address bar), Ctrl+A (select all), Ctrl+C (copy). It then reads the clipboard, restores your original clipboard content, and presses Escape to return focus to the page. This takes under one second and is transparent to the user.

For Firefox, the addon automatically retries up to three times with increasing delays to account for Firefox's slower address-bar focus behaviour.

Clipboard restore: URL Announcer always saves and restores your clipboard content by default. If you disable this in Settings, the URL replaces whatever was in your clipboard.

Installation

  1. Download urlAnnouncer-3.0.0.nvda-addon.
  2. Double-click the file. NVDA will ask to install it.
  3. Click Yes.
  4. Restart NVDA when prompted.
  5. Open any browser, navigate to a webpage, and press NVDA+Shift+U.

Changing the Entry Gesture

If NVDA+Shift+U conflicts with another add-on on your system, you can change it: go to NVDA Menu → Preferences → Input Gestures → URL Announcer and assign any gesture you prefer.
